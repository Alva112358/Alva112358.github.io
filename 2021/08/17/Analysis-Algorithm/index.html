Master Theorem

For given formula with \(a \geq 1\), \(b >1\). \[ \begin{align*} & T(n) = aT(\frac{n}{b})+f(n) & \end{align*} \]

1. Root Dominate

If \(f(n)=O(n^{\log_b{a}-\varepsilon})\), then \(T(n)=\Theta(n^{\log_b{a}})\).

2. Balanced

If \(f(n) = \Theta(n^{\log_b{a}})\), then \(T(n)=\Theta(n^{\log_ba}\log n)\).

3. Leaves Dominate

If \(f(n)=\Omega(n^{\log_b a-\varepsilon})\), then \(T(n)=\Theta(f(n))\).

Extended Master Theorem of Case 2

If \(f(n)=\Theta(n^{\log_b a}\log^{k}n)\), then \(T(n)=\Theta(n^{\log_ba}\log^{k+1}n)\).
