LeetCode 927 Three Equal Parts

题目描述

题目链接：https://leetcode.com/problems/three-equal-parts/

You are given an array arr which consists of only zeros and ones, divide the array into three non-empty parts such that all of these parts represent the same binary value.

If it is possible, return any [i, j] with i + 1 < j, such that:

  • arr[0], arr[1], ..., arr[i] is the first part,
  • arr[i + 1], arr[i + 2], ..., arr[j - 1] is the second part, and
  • arr[j], arr[j + 1], ..., arr[arr.length - 1] is the third part.
  • All three parts have equal binary values.

If it is not possible, return [-1, -1].

Note that the entire part is used when considering what binary value it represents. For example, [1,1,0] represents 6 in decimal, not 3. Also, leading zeros are allowed, so [0,1,1] and [1,1] represent the same value.

Constraints:

  • 3 <= arr.length <= 3 * 104
  • arr[i] is 0 or 1

题目要求找到满足限制条件的下表 $i$ $j$，使得 $[0,i-1]$，$[i, j]$，$[j+1, n-1]$ 的二进制表示的值相同。

解决方案

可以将问题转换为相同的 1 的个数：即三个部分的二进制表示应该拥有相同个数的 1，并且每一部分的首个 1 以后出现的数字顺序相同，即可验证相等，写成过程的表达如下所示：

  1. 首先需要计算数组中 1 出现的次数 $c$；
    • 如果 $c = 0$，则数组可以表示成相等的三部分，并且每部分的十进制值都是 0，此时只需要返回 $[0,\text{arr.size()}-1]$.
    • 如果 $\text{mod}(c, 3) \neq 0$，则不可能表示成拥有相同 1 的三部分，因此返回结果 $[-1,-1]$.
  2. 遍历数组，记第 1 次出现 1 的下标未 $\text{start}$，第 $c/3+1$ 次出现的 1 的下标未 $\text{mid}$，第 $2c/3+1$ 次出现的 1 的下标为 $\text{end}$.
  3. 验证 $[\text{start},\text{mid}-1]$，$[\text{mid},\text{end}-1]$，$[\text{end},\text{arr.size()-1}]$ 相同。
